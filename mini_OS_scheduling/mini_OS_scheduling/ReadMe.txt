========================================================================
    콘솔 응용 프로그램 : mini_OS_scheduling 프로젝트 개요
========================================================================

응용 프로그램 마법사에서 이 mini_OS_scheduling 응용 프로그램을 만들었습니다.

mini_OS_scheduling 응용 프로그램을 구성하는 각 파일에 대한
요약 설명이 포함되어 있습니다.


mini_OS_scheduling.vcxproj
    응용 프로그램 마법사를 사용하여 생성한 VC++ 프로젝트의 기본 프로젝트 파일입니다. 파일을 생성한 Visual C++ 버전에 대한 정보와 응용 프로그램 마법사를 사용하여 선택한 플랫폼, 구성 및 프로젝트 기능에 대한 정보가 포함되어 있습니다.

mini_OS_scheduling.vcxproj.filters
    응용 프로그램 마법사를 사용하여 생성된 VC++ 프로젝트의 필터 파일입니다. 이 파일에는 프로젝트의 파일과 필터 간의 연결 정보가 들어 있습니다. 이러한 연결은 특정 노드에서 유사한 확장명으로 그룹화된 파일을 표시하기 위해 IDE에서 사용됩니다. 예를 들어 ".cpp" 파일은 "소스 파일" 필터와 연결되어 있습니다.

main.cpp
    기본 응용 프로그램 소스 파일입니다. 이 파일에는 main 함수가 포함됩니다.

memoryManagement.cpp
	기본 응용 프로그램 소스 파일입니다. 이 파일은 임의의 메모리를 만들고 일정 개수의 Thread를 생성하여 
	OS level의 Memory management를 구현함을 목적으로 합니다.
	임의의 Page를 만들어 각각의 해당하는 Thread의 Logical Address를 Physical Address로 변경하고
	Double Linked List를 사용하여 Memory 할당 함수를 구현하고
	이를 통한 할당 전, 후 상태를 출력합니다. 

scheduling.cpp
	이 파일은 OS level의 System Scheduling을 구현한 것으로서
	간단하게 우선순위를 가지는 3개의 Queue를 만들고 일정 개수의 Thread(여기서는 15개)를 만들어
	Random하게 Priority를 부여합니다. 이 후 Priority의 Level에 따라 해당 Thread를
	High Que, Middle Que, Low Que로 분류하고 
	일정 시간을 간격으로 출력하게 합니다. 이 때 높은 우선순위를 가지는 Thread가 
	상대적으로 자주 출력되게 하였습니다.
